# Professional QR Code Generator

A sleek, glassmorphic web application to generate high-quality QR codes with support for custom redundancy levels and logo embedding.

![Logo](https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/solid/qrcode.svg)

## 🚀 Features

-   **Instant Generation**: Convert any text or URL to a QR code in real-time.
-   **Custom Redundancy**: Choose between Low (7%), Medium (15%), Quartile (25%), and High (30%) error correction levels.
-   **Logo Support**: Upload your own image (logo) to be placed in the center of the QR code.
-   **Auto-Redundancy**: Automatically switches to High redundancy when a logo is added to ensure scannability.
-   **High Resolution Downloads**: Download your QR code as a 1024x1024 PNG file.
-   **Glassmorphic Design**: Modern, responsive UI with a premium aesthetic.

## 🛠️ Technology Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Modern styling with CSS variables, Flexbox/Grid, and Backdrop-filters.
-   **JavaScript (ES6+)**: Core application logic.
-   **[qr-code-styling](https://github.com/kozakdenys/qr-code-styling)**: Professional QR code rendering library.

## 📥 Getting Started

To run this application locally, simply:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/text-to-qr-code-generator.git
    ```
2.  **Open the app**:
    Launch `index.html` in any modern web browser (e.g., Chrome, Firefox, Opera GX).

## 🧩 Usage

1.  Enter your text or URL in the input field.
2.  (Optional) Select your desired Correction Level.
3.  (Optional) Click the **"+"** button to upload a logo image.
4.  Click **"Generate QR"**.
5.  Click **"Download"** to save your QR code as a PNG.
